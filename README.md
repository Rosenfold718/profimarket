# ProfiMarket

Маркетплейс профессионалов для строительной отрасли — площадка для заказчиков и исполнителей (сметчики, эксперты, юристы, проектировщики, кадастровые инженеры).

## Стек

- **Next.js 16** (App Router, TypeScript)
- **Tailwind CSS 4** + **shadcn/ui**
- **Prisma** ORM (SQLite)
- **Socket.io** (реальные чаты)
- **Zustand** (состояние клиента)
- **Framer Motion** (анимации)
- **jose** (JWT)

## Установка

```bash
# Установить зависимости
bun install

# Настроить переменные окружения
cp .env.example .env

# Создать БД и применить схему
bun run db:push

# (Опционально) Заполнить тестовыми данными
bun run prisma/seed-extra.ts

# Запустить
bun run dev
```

## Функциональность

- 🔐 Авторизация (регистрация / вход) с JWT
- 📋 Каталог заказов с фильтрами и поиском
- 💬 Чаты в рамках заказов (Socket.io)
- 👤 Профили пользователей с аватарами
- 📝 Отклики на заказы
- 📊 Панель управления с аналитикой
- 📱 Адаптивный дизайн (мобильная + десктопная версия)

## Структура

```
src/
  app/api/          — API-маршруты
  components/app/   — Шапка, сайдбар, тосты
  components/ui/    — shadcn/ui компоненты
  hooks/            — Кастомные хуки
  lib/              — Утилиты (auth, db, fetch)
  stores/           — Zustand хранилище
  views/            — Страницы приложения
mini-services/      — Socket.io чат-сервис
prisma/             — Схема БД и сиды
```
